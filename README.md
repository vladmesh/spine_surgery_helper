# Spine Surgery Helper

Десктопное приложение (Tkinter) для хирургов-вертебрологов. Считает восстановление исходной анатомии позвоночника по измерениям высот тел позвонков и межпозвоночных дисков, ведёт карточки пациентов и формирует отчёты.

## Возможности

- Ввод параметров позвонков и дисков (КТ, интраоперационный контроль).
- Расчёт восстановления исходной анатомии.
- Хранение данных пациентов в SQLite.
- Экспорт отчётов в Excel (xlsxwriter).
- Отправка отчёта на почту по SMTP.

## Запуск

```bash
pip install -r requirements.txt
cp .env.example .env   # заполнить EMAIL и EMAIL_PASSWORD
python main.py
```

## Сборка

Исполняемый файл собирается через PyInstaller:

```bash
pyinstaller main.spec
```

## Конфигурация

`.env` (в git не коммитится):

- `EMAIL` — почта отправителя.
- `EMAIL_PASSWORD` — app-пароль для SMTP (Яндекс).
